
# Contribuer à AfroMarket

1. Crée une **User Story** via l’Issue Form dédié.
2. Estime en Story Points et assigne un **milestone**.
3. Branche `feature/US-xxx-titre-court`.
4. Ouvre une PR liée ("Closes #ID").

## Qualité
- CI verte, lint + tests, couverture ≥ 80%
- Observabilité (logs, metrics, traces)
- Sécurité (validation inputs, authN/Z, rate-limit)

## Commits
Conventional Commits: feat/fix/perf/refactor/test/docs/ci/chore
